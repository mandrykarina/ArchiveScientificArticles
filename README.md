# База научных статей

Единая система для *сбора, обработки, хранения и поиска* научных публикаций из нескольких источников.

## Кратко о проекте

Проект решает задачу объединения научных статей из разных платформ в одной системе.  
Публикации автоматически собираются, проверяются на дубликаты, переводятся, очищаются, нормализуются и сохраняются в базу данных.  
Далее пользователь может работать с ними через веб-интерфейс: искать, просматривать и переходить по категориям.

## Почему это важно

Сегодня научные статьи:
- распределены по разным платформам
- имеют неоднородные метаданные
- неудобны для единого поиска и анализа

Наш проект превращает разрозненные публикации в *единую структурированную систему*.

## Что умеет система

- собирать статьи из нескольких научных источников
- работать в режиме разовой выгрузки и мониторинга обновлений
- исключать повторную обработку дубликатов
- переводить данные на русский язык
- очищать и нормализовать метаданные
- извлекать ключевые слова
- сохранять данные в БД
- предоставлять удобный веб-интерфейс для поиска и навигации

## Источники

- *arXiv* — международный архив препринтов
- *CyberLeninka* — российская научная платформа
- *ChinaXiv* — китайский архив публикаций

## Архитектура

*Источники → Парсинг → Обработка → База данных → Сайт*

## Технологии

*Backend и обработка*
- Python
- YandexGPT
- MarianMT
- langdetect
- YAKE
- pymorphy3

*Веб-часть и хранение*
- Flask
- Jinja2
- SQLAlchemy
- PostgreSQL / SQLite
